CF954I Yet Another String Matching Problem
2019-12-11|OI
字数总计: 676|阅读时长: 3 分钟

链接

题解为什么都是 FFT ~~~

思路来自于 Frame\color{black}{\textbf{F}}\color{red}{\textbf{rame}}Frame 巨佬

题面翻译：对于两个等长的字符串 s,ts,ts,t ，每次可以花费 111 的代价，将某一种字符全都替换成另一种字符。

设 f(s,t)f(s,t)f(s,t) 为最少花费多少代价，使得 s=ts=ts=t 。

给定字符串 s,ts,ts,t ，求 sss 每个与 ttt 长度相同的字串 s′s's′ 的 f(s′,t)f(s',t)f(s′,t) 。

容易发现，将字符 iii 改成字符 jjj ，与将字符 jjj 改成字符 iii 效果相同，所以当我们修改时，使 i≤ji\le ji≤j 。

还能发现，将 iii 修改为 jjj ，再将所有的 jjj 修改为 kkk ，与 i,ji,ji,j 都修改为 kkk 效果相同。

于是我们可以 O(6!)O(6!)O(6!) 枚举每个字符 iii 将要修改为的字符 j(i≤j)j(i\le j)j(i≤j) ，并对 s,ts,ts,t 串进行 O(n)O(n)O(n) 的 KMP 。对于能够匹配的字串，更新答案。
